Membership USBankPay Payment Gateway for WooCommerce; start accepting payments on your store. 

== Description ==
Membership USBankPay payment gateway plugin for WooCommerce. Start accepting payments on your store.

== Installation ==

Pre-requisite

1. To add Membership USBankPay payment gateway to your application and start collecting payments, you will first need to contact Membership USBankPay
2. Get following things from Membership USBankPay
    - Merchant ID
You will need these values when setting up this plugin.

Setup

1. Upload the plugin folder to the ‘/wp-content/plugins/’ directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. From the admin site, go to Woocommerce > Settings and select the Checkout tab or Payments tab depending on your WordPress version. 
4. Then click on Membership USBankPay from the list of available payment methods. If you don't see it here, make sure you have activated the plugin.
5. Once on the Membership USBankPay settings page, fill in the details received from bank.
